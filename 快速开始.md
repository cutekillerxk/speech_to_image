# 快速开始指南

## 前置要求

- Node.js 16+ 
- npm 或 yarn
- 豆包API密钥（可选，不配置也能运行测试模式）

## 一键启动

### Windows (PowerShell)

```powershell
# 1. 安装后端依赖
cd backend
npm install

# 2. 配置环境变量（可选）
# 复制 .env.example 为 .env 并填写API密钥
# 如果没有API密钥，可以跳过此步骤，系统会使用模拟API

# 3. 启动后端服务
npm start

# 4. 新开一个终端，安装前端依赖
cd frontend
npm install

# 5. 启动前端服务
npm start
```

### Mac/Linux

```bash
# 1. 安装后端依赖
cd backend && npm install

# 2. 配置环境变量（可选）
cp .env.example .env
# 编辑 .env 文件，填写你的API密钥

# 3. 启动后端服务（后台运行）
npm start &

# 4. 安装前端依赖
cd ../frontend && npm install

# 5. 启动前端服务
npm start
```

## 访问应用

打开浏览器访问：http://localhost:3000

## 测试流程

1. **点击"开始录音"** - 允许浏览器访问麦克风
2. **说话** - 描述你想要生成的图片（例如："一只可爱的小猫"）
3. **点击"停止录音"** - 音频将发送到后端处理
4. **等待处理** - 系统会先识别语音，再生成图片
5. **查看结果** - 图片会自动显示
6. **浏览历史** - 使用"上一张"/"下一张"按钮查看历史记录

## 故障排查

### 后端无法启动
- 检查端口3001是否被占用
- 检查Node.js版本是否 >= 16
- 查看控制台错误信息

### 前端无法启动
- 检查端口3000是否被占用
- 确认后端服务已启动
- 检查网络连接

### 无法录音
- 检查浏览器是否允许麦克风权限
- 确认使用HTTPS或localhost（某些浏览器要求）
- 检查麦克风设备是否正常

### API调用失败
- 检查 `.env` 文件中的API密钥是否正确
- 查看后端控制台的错误日志
- 如果没有API密钥，系统会使用模拟模式（返回占位图片）
